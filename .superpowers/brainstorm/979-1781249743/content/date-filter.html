A — Быстрые кнопки

Один клик: Сегодня / 7 дней / 30 дней / Всё время. Просто и быстро, без ввода дат вручную.

📅 от: 01.05.2026
→
📅 до: 12.06.2026

B — Два поля «от / до»

Два текстовых поля (тип date) для ввода начальной и конечной даты. Точный контроль над диапазоном.

7 дней

C — Комбо (кнопки + свой диапазон)

Быстрые кнопки + кнопка «Свой» раскрывает поля ввода дат. Лучший баланс удобства и гибкости.

💡 Фильтр будет работать на всех трёх вкладках (Обзор, Сотрудники, Точки)
