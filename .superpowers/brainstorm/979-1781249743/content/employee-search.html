A

Поиск в шапке фильтров

Поле поиска встроено прямо в панель фильтров рядом с выпадающими списками. Фильтрует карточки и таблицу сотрудников в реальном времени по мере ввода.

B

Поиск над таблицей

Отдельная широкая строка поиска прямо над таблицей записей с ФИО. Фильтрует только таблицу, карточки остаются нетронутыми.

C

Единый поиск — везде

Одно поле в фильтрах, которое фильтрует сразу: карточки сотрудников, таблицу записей и бар-чарт «Топ-15 нарушителей». Поиск работает по имени и должности.
