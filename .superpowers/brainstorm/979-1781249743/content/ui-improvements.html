Улучшения UI — что хотим изменить?

Выбери все направления, которые хочешь улучшить

1

KPI-карточки

Анимированный счётчик уже есть. Добавить: иконки, трендовые стрелки (↑↓), цветной акцент-бордер снизу. Карточки станут информативнее с первого взгляда.

2

Таблица данных (DataTable)

Сейчас — простой список. Улучшить: чередующийся фон строк, sticky-заголовок при прокрутке, сортировка по колонкам, выделение строки при наведении.

3

Карточки сотрудников и точек

Добавить badge с топ-категорией нарушения, мини-прогресс-бар по количеству, плавный hover-эффект с подъёмом. Карточки станут нагляднее.

4

Панель фильтров

Текущие select-элементы заменить на стилизованные dropdown с иконками. Добавить счётчик активных фильтров, кнопку «Сбросить» всегда видимой в углу панели.
